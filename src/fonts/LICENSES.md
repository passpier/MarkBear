| Family | Source | Files |
|---|---|---|
| JetBrains Mono | https://github.com/JetBrains/JetBrainsMono | `jetbrains-mono*.woff2` |
| Fira Code | https://github.com/tonsky/FiraCode | `fira-code.woff2` |
| Cascadia Code | https://github.com/microsoft/cascadia-code | `cascadia-code-*.woff2` |

woff2 builds for JetBrains Mono, Fira Code, Cascadia Code, and Source Code Pro
were sourced from the [Fontsource](https://fontsource.org/) project (itself
redistributing the same upstream OFL releases as static/variable web fonts).
Hack's woff2 build was taken directly from its official
[v3.003 GitHub release](https://github.com/source-foundry/Hack/releases/tag/v3.003)
(`Hack-v3.003-webfonts.zip`).
